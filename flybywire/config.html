This page allows you to send data from a modern computer's mouse and keyboard over a USB cable to a locally connected Hootswitch, useful for situations where you don't want to use a period keyboard/mouse (workbench computers, constrained desks, development systems, etc). To use, connect the Hootswitch to your system via a USB cable then click Connect above.

Several modern web technologies are leveraged here, including the Web Serial API, Keyboard Lock API (TODO), and Pointer Lock API. Everything stays inside your browser, no inputs are passed to the Internet.

At least as of September 2024 most of these APIs are experimental and only have support on Chrome/Chromium and the various web browsers using that engine.
